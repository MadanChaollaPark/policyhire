# PolicyHire

A polished EU affairs jobs board prototype built as a feature-parity competitor to EUJobs.co, EuroBrussels, Jobs in Brussels, and Euractiv Jobs.

## What It Includes

- Advanced job search across keyword, location, policy area, category, seniority, contract, work mode, language, salary signal, and deadline.
- Two-pane desktop job browsing with verified employer signals, save actions, salary badges, deadline countdowns, and detailed role pages.
- Employer directory with organisation type, open roles, policy areas, salary transparency score, hiring velocity, and badges.
- Lobbying intelligence view for transparency-register style entities and Brussels Best style supplier categories.
- Fair Pay explorer with salary bands by role, city, seniority, and sample size.
- Recruiter pricing, campaign analytics, moderation queue, and guided-posting quality signals.
- Internship calendar, job alerts, career guides, SEO metadata, favicon, manifest, robots, and sitemap.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Notes

This is an original implementation with original branding and generated visual assets. It targets feature parity and competitive UI patterns without copying competitor content or protected branding.
