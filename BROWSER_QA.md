# Browser QA Matrix

## Required Viewports

- 1440x900 desktop: full hero, stats, tabs, three-column jobs layout, sticky detail pane.
- 1280x800 desktop: search results, employer directory, recruiter pricing, salary explorer.
- 1024x768 tablet landscape: hidden sidebar filters, visible mobile filter action, readable detail pane.
- 768x1024 tablet portrait: one-column section grids, no clipped cards.
- 390x844 mobile: stacked hero search, horizontal tab rail, one-column jobs, drawer filters.
- 360x800 mobile: no horizontal overflow, compact header, readable buttons and salary labels.

## Interaction Checks

- Search by keyword, location, policy area, category, seniority, contract, work mode, and language.
- Sort by relevance, salary high to low, and deadline soon.
- Save and unsave jobs.
- Open, apply, reset, backdrop-close, and Escape-close the mobile filter drawer.
- Switch through jobs, companies, lobbying intelligence, Fair Pay, recruiters, and guides.
- Verify focus states on header buttons, tabs, filters, result cards, save buttons, and CTAs.
